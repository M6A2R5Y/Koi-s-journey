# Mary Kinyanjui | Portfolio & Dev-Dashboard

Welcome to the **Control Center** – a premium developer portfolio and interactive dashboard built to showcase the academic background, professional experience, and technical skill set of Mary Kinyanjui.

Mary is a B.Sc. Mathematics & Computer Science graduate, Customer Operations expert, and aspiring Software Engineer & Cybersecurity analyst.

---

## 🚀 Key Features

This portfolio is structured as a dynamic, command-center style dashboard featuring:

1. **Dashboard (Overview)**: High-level overview of academic accomplishments, core career tracking (operations and support), and interest areas in software engineering and cybersecurity.
2. **Work Experience**: A detailed career track detailing professional roles, including operations and support at Teleperformance EPZ Majorel.
3. **NFC Flow Simulator**: An interactive simulator illustrating safe NFC/RFID data exchanges (from initial signal detection to sector decryption, database record matching, and secure ledger logging).
4. **Algorithms Visualizer**: A visual implementation of fundamental computer science sorting algorithms (Bubble Sort) highlighting key comparison and swap mechanics in real-time.
5. **Skills & Profile**: Highlights of core strengths, languages, development tools, and contact telemetry.

---

## 🛠️ Technology Stack

- **Framework**: [React](https://react.dev/) (v18)
- **Bundler & Build Tool**: [Vite](https://vite.dev/)
- **Programming Language**: [TypeScript](https://www.typescriptlang.org/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Styling**: Modern, responsive CSS with modular glassmorphism layout, variable themes (Light/Dark mode support), and micro-animations.

## 📦 Getting Started

Follow these steps to run the portfolio dashboard locally on your system.

### Prerequisites

Ensure you have [Node.js](https://nodejs.org/) installed.

### Installation

1. Clone or download the repository:
   ```bash
   cd "Koi's journey"
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the local development server:
   ```bash
   npm run dev
   ```

4. Build the application for production:
   ```bash
   npm run build
   ```
